# Mars Tactics

Welcome to the official website repository for Mars Tactics, a turn-based tactical strategy game set on the war-torn red planet of Mars.

## Overview

This repository contains the website code for Mars Tactics, featuring:
- Latest news and updates
- Media gallery with screenshots and videos
- Comprehensive game guides
- Legal documentation

## Structure

The website includes the following main sections:
- `/` - Home page with game introduction, features, screenshots, system requirements, and FAQ
- `/news` - Latest updates and announcements from IGN and other sources
- `/media` - Screenshots, videos, and downloadable wallpapers
- `/guides` - Comprehensive game guides covering tactical combat, strategic layer, factions, and character progression
- `/privacy-policy` - Privacy policy documentation
- `/terms-of-service` - Terms of service documentation

## Development

### Prerequisites
- Static HTML/CSS/JS website
- No build process required
- Deploy directly to Cloudflare Pages

### Setup
1. Clone the repository
2. Open `index.html` in a web browser or deploy to Cloudflare Pages
3. All static assets are in the `static/` directory

## Features

- **Responsive Design**: Mobile-friendly layout
- **SEO Optimized**: Complete meta tags, structured data, and sitemap
- **Military Dark Theme**: Deep gray, black, and orange color scheme
- **Version Control**: CSS/JS files use version numbers for cache busting
